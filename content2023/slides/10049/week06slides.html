Midterm

  1. Written online, starting at the beginning of class in Week 7.
  2. Length max. 90 minutes, SLATE Quiz with Respondus lockdown.
  3. You must physically attend the class and present your Student ID card.
  4. Covers concepts and skills related to the learning objectives (next slide)
  5. Be prepared to demonstrate your mastery
    • of all material from Weeks 1 through 6 (Worksheets 1-5)
    • by answering questions in form of multiple-choice (MC) , true-false (F/T), multiple-select (MS), fill-in-blanks (FIB), matching (MAT), ordering (ORD) and coding/written answers (WR).
    • approx. 15 MC two points each , approx. 20 F/T one point each, 50 total,
      10-12 MS, FIB, MAT, ORD, WR questions, 3-6 points each, 50 total.

SYST10049 Web Development

Learning Outcomes

  1. Create a properly structured web page (HTML5)
  2. Represent the Document Object Model (DOM) in a hierarchical diagram.
  3. Recognize the importance of browser compatibility with standards.
  4. Style a web page (CSS3)
  5. Create page layouts using media queries.
  6. Use internet protocols to publish a web site
  7. Explain how web servers display web pages.
  8. Describe how to secure a web site.
  9. Use design techniques to layout a user-friendly web site.
  10. Design forms that use major controls and form handling logic.
  11. Create Use simple client-side scripts with JavaScript.

What notes can I use during exam?
  • You are allowed to use YOUR OWN NOTES. There is no set limit.
  • If you are making your notes electronically, print them out and organize in a binder.
  • If you are making handwritten notes, bring your notebook or binder.
  • Printouts of other sites and publications are not allowed.
  • Make sure to organize your notes so that you do not distract your neighbors!
  • YOU ARE NOT ALLOWED TO SHARE YOUR NOTES WITH OTHERS!

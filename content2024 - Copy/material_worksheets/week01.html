3.2. Learn about how the web works
Read, study, and make notes
  1. How the web works (MDN web docs)
  2. How does the Internet work? (MDN web docs)
  3. Client-server environment
  4. Internet: the backbone of the Web
  5. Order in which component files are parsed
  6. TCP/IP Protocol Suite
  7. How the web works
  1. Navigation
  2. Fetching resources
  3. Parsing HTML and CSS
  4. Executing JS
  5. Rendering
Test yourself: formulate your own answers
  1. Describe the web ecosystem. What components comprise this ecosystem? What are the major computer components and how they work?
  2. What is the purpose of a web browser? Describe what happens when a user requests a web page.
  3. Describe how the browser processes an HTML document. Is there a difference in the process when the file is retrieved using FILE protocol and HTTP protocol?
  4. Describe what happens when you enter "sheridancollege.ca" in the browser's address area. What is the name of the file that you are requesting?
3.3. Learn about URLs
Read, study, and make notes
  1. What is a URL? (MDN web docs)
  2. Uniform Resource Locator (URL)
Test yourself: formulate your own answers
  1. What is the basic anatomy of a URL? How do you use URLs? Compare absolute and relative URSs. What is a semantic URL?
  2. Given the following URL , identify each part
    https://www.testing.server:8080/projects/trees/index.html
    1. Name of file being requested
    2. first directory name in the path
    3. last directory name in the path
    4. Authority
    5. Protocol
    6. Port
    7. Server name
    8. Path to file

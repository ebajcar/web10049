To connect a set of CSS rules to an HTML document, use one of the following: external, internal, or inline.

External Style Sheets

Internal (Embedded) Styles

Inline Styles

Using the inline method outside of JavaScript violates responsive design and separation of languages (duties!) Each declaration would have to be applied to every element that needs the property value changed within the page.

Normal flow (defaults)

Each browser sets the properties to default values. Refer to individual vendor's documentation and to W3C specifications; vendor's conform to industry standards, but will have variations. It is important to understand how browser applications construct the DOM and CSSOM to gain insight how CSS rules are applied.

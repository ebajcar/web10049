Project Brief
  1. Read these instructions in its entirety. You are expected to follow these instructions where they vary from the MDN instructions.
  2. Read actively and make notes information about the project and related HTML elements Structuring planet data (MDN)
  3. Create a new project file structure (check previous Worksheets) in projects subdirectory. The project name is my_planets. .
  4. The home page has no content in the MAIN element.
  5. The planets.html is a secondary page that contains the data table in the MAIN element.
  6. The navigation, header, and footer are common to all pages on this site. For now we just have Home (index0.html) and Planets (content/planets.html) links.
  7. Start with a copy of the standard site_template project from previous weeks. (MDN's blank-template.html is missing several items that our site template uses!)
  8. Download the minimal-table.css file to your css folder.
  9. Add the link
    <link href="minimal-table.css" rel="stylesheet" type="text/css">
    to the bottom of the HEAD section, just before the </head>
  10. Add raw data to the MAIN element from the planets-data.txt file.
  11. Mark up the raw data. The given css rules will format your table when you choose the correct elements. Don't forget to validate regularly.

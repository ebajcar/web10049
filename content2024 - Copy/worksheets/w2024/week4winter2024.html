Block and inline flow
Read further Block-level elements (MDN web docs) By default, block-level elements begin on new lines.
Read further Inline elements (MDN web docs) By default, inline elements stay in line of the parent element. To determine what content category a particular element belongs to, examine the specifications document.

2. Read and explore more concepts

Activity 1. Complete tutorials Page structure (WCAG Success criteria: 1.3.1., 2.4.1, 2.4.6, 2.4.10)
  • Page Regions Page header, page footer, navigation, main content, complementary contant, using WAI-ARIA. (Techniques: ARIA11)
  • Headings Heading ranks, organize passages of text, headings that reflect the page organization. (Techniques: ARIA12, G130, G141, H42, H69, H80, PDF9)
  • Labeling Regions Using area-labelledby and aria-label. (Techniques: ARIA11, H69)
  • Content Structure Articles, sections, paragraphs, lists, quotes, figures, images, tables. (Techniques: G62, H40, H48, H49)
References (HTML Living Standard)
  • 3.2.5 Content models The "nothing" content model. Kinds of content (Metadata, Flow, Sectioning, Heading. Phrasing, Embedded, Interactive). Paragraphs.
Activity 2. Hyperlinks (HTML Living Standard)
  • 4.6 Links
  • Hyperlinks links to other resources that are generally exposed to the user by the user agent so that the user can cause the user agent to navigate to those resources, e.g. to visit them in a browser or download them.
Activity 3. Complete MDN tutorials
  • Creating hyperlinks • what is a hyperlink? • anatomy of a link • block-level links • image links • adding supporting information with the title attribute • quick primer on URLs and paths • Document fragments • Absolute versus relative URLs • use clear link wording • linking to non-HTML resources — leave clear signposts • use the download attribute when linking to a download
  • Active learning: creating a navigation menu
Complete tutorials Menus (MCAG Success criteria: 2.4.1, 2.4.3, 3.4.5, 2.4.7, 2.4.8, 4.1.2)
Activity 4. Complete tutorials Menu Structure (Web Accessibility Initiative - WAI)
Menu represenation, identify menus, label menus, indicate the current item. (ARIA6, ARIA11, G14, G63, G65, G127, G128, G161, G182, G183, H4, SCR26)

Web browser

  1. SYST10049
  2. Web environment
  3. Web browser…

A web browser is an application software used to access the web. Web browsers understand HTML, CSS, and Javascript and render the web pages to the visitors. Web browsers, such as Netscape, Opera, and Edge contact remote web servers and make requests on behalf of users. The documents can contain hyperlink points (links) to other documents, which may or may not live on the server that the user originally contacted.

The purpose of a web browser is to get content from the web and display it on the user's device. Browser requests each resources from the web server(s) using the information in the URL (protocol, domain, path, file)

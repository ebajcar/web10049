Activity 7.3. PRACTICE problem solving and coding in PHP

7.3.1. Problem —Write a program to play a game of Casino Craps.
  1. Use the modified Javascript solution Click here and convert the solution to PHP.
  2. Compare Javascript and PHP solutions and implement.
  3. Refactor both versions (using switch and nested ifs) to use switch instead of nested ifs
  4. Refactor as fully-functional or object-oriented solution, with appropriate user input.
  5. Add a simple score-keeping algorithm.
7.3.2. Problem —Write a program to play a game of Rock-Paper-Scissors-Lizard-Spock.
  1. Use the modified Javascript solution of Rock-Paper-Scissors (both players random selections, no user input)
    Click here and convert the solution to PHP.
  2. Use the modified Javascript solution of Rock-Paper-Scissors-Lizard-Spock (both players random selections, no user input)
    Click here and convert the solution to PHP.
  3. Modify the PHP version to use an imput to capture user's selection (Hint: use GET method)
  4. Add a simple score-keeping algorithm. (For example, a scheme such as a winner of five consecutive games earns a point.)
7.3.3. Problem —Write a program to play a game Tic Tac Toe
  1. Use the modified Javascript solution Click here and convert the solution to PHP. Note this solution will have NO Javascript code.
  2. Refactor as fully-functional or object-oriented code. Note this solution will have NO Javascript code.

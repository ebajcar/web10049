9.2.4. PHP Data Objects (PDO) Class
PHP Data Objects (PDO) is a data-aware class —PHP Manual
  • a class connects to a database
  • the class becomes the data source
  • the form (business object) that uses the data becomes the data consumer
In this course, we use PDO only! (Why? Compare to MySQLi)
  • PDO connections are established by creating instances of the PDO base class.a database
  • It doesn't matter which driver you want to use; you always use the PDO class name.
  • The constructor accepts parameters for specifying the database source (known as the DSN) and optionally for the username and password (if any).
  • PHP Connect to MySQL @w3schools
General process
  • Design and construct user interface - HTML form.
  • Design, build and maintain a database.
  • Connect to data file.
  • Send an SQL query.
  • Process retrieved data.
  • Close connection to data file.

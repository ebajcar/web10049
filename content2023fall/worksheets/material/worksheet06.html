Activity 6.1. Review and preparation

Complete, review. dig deeper Worksheets 1, 2, 3, 4, and 5. Catchup from previous weeks.

Revisit the Class Plan Information and read about Active learning, Learning course content, Learning for mastery, not grades, How evaluation works, Formative & Summative Evaluation, etc.

It is important to continue working with Javascript. Practice regularly. Refactor code, explore different techniques to do the same thing, learn to evaluate your own work and identify how you can improve. Read and explore alternate problems and potential solution, practice communicating and collaborating with others. Write the same solution in both fully functional paradigm AND fully object-oriented paradigm.

Activity 6.3. PRACTICE problem solving and coding in PHP

Create a new project called mathWorks with standard file organization (content, css, images, js) and home page (index.html) with appropriate navigation to secondary pages.

6.3.1. Problem —Write a program that adds two numbers and displays result together with the two values provided by the user. Implement as a secondary page in the mathworks project.

6.3.2. Problem —Write a program that will allow the user to guess a number between 1 and 10 that was randomly chosen by the program. Implement as a secondary page in the mathworks project.

6.3.3. Problem —Write a program that calculates a given degrees Celsius in degrees Fahrenheit and displays a phrase that includes both values. Implement as a secondary page in the mathworks project.

6.3.4. Problem —Write a program to calculate roots of a quadratic equation. Implement as a secondary page in the mathworks project.

6.3.5. Problem —Write a program to calculate area of a circle. Implement as a secondary page in the mathworks project.

6.3.6. Problem —Write a program to calculate Fibonacci numbers. Implement as a secondary page in the mathworks project.

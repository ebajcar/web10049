Elements and attributes
when you mark up text with the P element, you are creating the "paragraph object" with its properties, methods, and events. Some of the properties are fg and bg colours, size of individual letters, line height of the paragraph, space before and after the paragraph, etc.
when you mark up an image with the IMG element, you are creating the "image object" with its proeprties, methods, and events. Some of the properties are the size of the image (height and width). It does not have properties like size of individual letters or line height though.
there are also differences in behaviour. A paragraph will start on a new line and will take the entire last line regardless how much it needs. The following element will start on a new line. a paragraph has a "block" flow behaviour. On the other hand, an image will not create a new line and other elements will continue on the same row if there is room and they also do not break a line. An image has an "inline" flow behaviour.

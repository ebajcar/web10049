p-red

A color can be specified as an RGB value, using this formula rgb(%red, %green, %blue).

.p-red { background-color: rgb(100%, 0%, 0%); }
d-red

A color can be specified as an RGB value, using this formula rgb(decimal red, decimal green, decimal blue).

.d-red { background-color: rgb(255, 0, 0); }
x-red

A color can be specified as an RGB value, using hexadecimal numbers.

.x-red { background-color: #ff0000; }
t-red

A color can be specified as an RGBA value, using ...

.t-red { background-color: rgba(255,0,0,0.65); }
h-red

A color can be specified as HSL value .

.h-red { background-color: hsl(0, 100%, 80%); }
th-red

A color can be specified as HSLA value, using hsla(hue, saturation, lightness, alpha) The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (fully opaque).

.th-red { background-color: hsla(0, 100%, 30%, 0.5); }

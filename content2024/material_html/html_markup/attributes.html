HTML Attributes

Global attributes

Global attributes are common to and may be specified on all HTML elements:

  • accesskey
  • contenteditable
  • dir
  • draggable
  • hidden
  • lang
  • spellcheck
  • style
  • tabindex
  • title
  • translate
  • ...and others.

DOM defines the user agent requirements for the class and id attributes for any element.

  • id
  • class

For full list and more details on each, refer to

  • 3.2.6 Global attributes section of the HTML Living Standard
  • Global attributes (W3C)

Attribute minimization

Rules: The term attribute minimization refers to an exception of HTML5 rule that single-valued attributes are minimized. The rules states that ALL attributes must have a quoted value assigned. In some cases, there is only one value that can be assigned to the attribute, such as <video controls="controls"> which in HTML5 only, is minimized to <video controls>

Element-specific attributes

Some elements have attributes that are specific to the element. For example, the <ol> element, by default starts with the natural number 1; you can change the starting value with the attribute start and the kind of marker to use with the attribute type. the code <ol start="3" type="a"> will produce a list with the first item numbered "c. First item", instead of the default "1. First item".

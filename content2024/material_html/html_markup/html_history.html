HTML A mark-up language

The goal of Web design is not merely to dazzle, but to deliver information to the widest audience possible... Given the powerful capabilities of modern graphical desktop browsers, it is now possible to provide a progressive, gradually enhanced experience across a wide array of browsers, using one markup document and a variety of different stylesheets, not selectively delivered to the user through browser sniffing, but rather requested by the client itself. Leave no one behind.
—Inclusive Web Design For the Future with Progressive Enhancement. Steven Champeon and Nick Finck

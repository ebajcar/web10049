CSS Specificity

Specificity is a measure of how specific a selector is â€” how many elements it could match.
Element selectors have low specificity.
Class selectors have a higher specificity, so will win against element selectors.
ID selectors have an even higher specificity, so will win against class selectors. The only way to win against an ID selector is to use !important.

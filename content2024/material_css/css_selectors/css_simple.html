Simple selectors

Simple selectors (select elements based on name, id, class) Examples:
universal selector * {} every element on the page
type (tag name) selector h1 {} all H1 elements on the page
class selector .my-class {} all elements with class attribute set <… class="my-class">
id selector #my-id {} all elements with id attribute set <… id="my-id">

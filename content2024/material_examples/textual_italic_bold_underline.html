Italic, bold, underline
The elements <b>, <i>, and <u> came about so people could write bold, italics, and underlined text in an era when CSS was still supported poorly or not at all. These elements only affect presentation and not semantics, are known as presentational elements and should no longer be used (Read more MDN's HTML text fundamentals, because semantics is so important to accessibility, SEO, etc. The concept of italics isn't very helpful to people using screen readers, or to people using a writing system other than the Latin alphabet.

  is a North American species of goose. Its name
  derives from the typically white plumage.
</p>

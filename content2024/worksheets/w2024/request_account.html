SYST10049 Web Development:Instructions

Request account credentials on web server

  1. Review what is a web server, file organization, etc.
  2. Requesting password

    What happens when you request a password? An account is created on the server. The account includes

    1. credentials (username and password)
    2. file storage on the server starting with your home directory (/home/username)
    3. webspace at /home/username/public_html
    4. web URL
    5. access to cPanel (a set of maintenance tools, including File Manager
  3. tech.fast.sheridanc.on.ca/ (Services > LAMP/Git). Information for Hostname, Web URL, cPanel URL, FileZilla setup, WinSCP setup, username and password info.
  4. Using credentials to access server with cPanel
  5. Using credentials to access server with FileZilla

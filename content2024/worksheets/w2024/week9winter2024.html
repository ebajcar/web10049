Activity 9.3. Practice. Practice. Practice.

Create a local project site called week9notes
Create a standard file organization in C:\public_html\syst10049. Use your home page as an overview and navigation to all the secondary pages. As you read and study the tutorials and modules in Activity 8.2. you come across examples. Implement each example as a secondary page. Use comments to cite where the example came from and annotate with your own observations, descriptions, and explanations. When you dig deeper and cross-reference to other sites, include notes on any connections and additional information you discovered. Use embedded styles (using STYLE element) for each page. Leave the home page unstyled. Validate HTML code, CSS code, accessibility. Practice publishing.
Create a local project site called biography
Create a standard file organization in C:\public_html\syst10049. Link to the project instructions "Styling a biography page" Download the starting point file and organize all files appropriately. This project has only one HTML page, the home page. Read through the Project Brief carefully and fully, then complete all the steps. Once you have completed, tested, and validated the single-file solution, create a new file in the css folder and name it bio.css. Move all style rules to the bio.css file and replace the style element with an appropriate link element. Retest and revalidate. Publish. Test for accessibility and correct as required.

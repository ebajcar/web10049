What is the difference between meter and progress?

Meter
The METER element defines a scalar measurement within a known range, or a fractional value.
attribute VALUE specifies the current value of the gauge. Required.
attribute HIGH specifies range that is considered to be a high value.
attribute LOW specifies range that is considered to be a low value.
Progress
The PROGRESS element represents the completion progress of a taxk
attribute MAX specifies how much work the task requires in total. Default value is 1.
attribute VALUE specifies how much of the task has been completed.

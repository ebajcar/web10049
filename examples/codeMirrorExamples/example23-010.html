Reminder: by the end of this course, you need to be able to demonstrate the learning outcomes

Course Learning Outcomes

  1. Create a properly structured web page using HTML5.
  2. Represent the Document Object Model (DOM) in a hierarchical diagram.
  3. Recognize the importance of browser compatibility with HTML5 and CSS3 standards.
  4. Style a web page using Cascading Style Sheets (CSS3).
  5. Publish a web page on the internet.
  6. Create page layouts that adapt to suit the constraints of the display using media queries.
  7. Explain how web servers display web pages.
  8. Secure a web page.
  9. Logically layout a web space.
  10. Design forms that use major controls and form handling logic.
  11. Create client-side form data validation.

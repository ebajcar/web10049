Publishing process

  1. Using FileZilla client (or another FTP application), connect to the server.
  2. Required credentials
    HOSTNAME dev.fast.sheridanc.on.ca USERNAME your Sheridan user name PASSWORD sent to your Outlook email on request PORT 22
  3. Upon successful connection, find the project you want to publish on your machine, and place yourself in the parent directory.
  4. Next, find the webspace on the server and find the directory where you want to upload your project.
  5. Drag and drop the entire project from your machine to the server.
  6. Using the browser, test your project by using the web URL (found at http://tech.fast.sheridanc.on.ca/services/lamp)

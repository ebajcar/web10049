Learning Outcomes

  1. Create a properly structured web page (HTML5)
  2. Represent the Document Object Model (DOM)
  3. Recognize browser compatibility with standards.
  4. Style a web page (CSS3)
  5. Create page layouts using media queries.
  6. Publish a web page on the internet.
  7. Explain how web servers display web pages.
  8. Secure a web page.
  9. Logically layout a web space.
  10. Design forms that use major controls and form handling logic.
  11. Create Use simple client-side scripts with JavaScript.

Preparing for Midterm

  1. Complete all homework (Worksheets 1-5).
  2. Study regularly and practice deliberately. Make connections to the Learning Outcomes.
  3. Make notes as you study and review your notes regularly.
  4. Join a Study Group, study together and test each other.

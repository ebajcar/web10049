Client-side Programming

Javascript (JS) in the browser

JavaScript (JS)

  • a lightweight, interpreted (just-in-time compiled), programming language with first-class functions
  • a prototype-based, multi-paradigm, single-threaded, dynamic language
  • supports object-oriented, imperative, and declarative styles

JavaScript (JS)

  1. Standards are
    • ECMAScript Language Specification (ECMA-262) and
    • ECMAScript Internationalization API specification (ECMA-402)
  2. Documentation throughout MDN is based on the latest draft versions of ECMA-262 and ECMA-402.

Inline Javascript

Obtrusive.

TO BE AVOIDED! 

<a onMouseOver="alert('you rolled over me');" 
    href="file.html" target="_blank">
  Move mouse over this text
</a>

Embedded Javascript

Good during development or
small amount of unique to-the-page code.

Add Javascript at the end of BODY element
using the <script> element 

<script>
/* declare and initialize variable txt to store a value */
txt = "Hello world!";
// display the stored value in variable txt at the console
console.log(txt);
</script>
</body>

External Javascript

Unobtrusive. Code reusability. Code maintenance.
Organize code in external files.

Add Javascript at the end of BODY element using the <script> element 

<script src="js/filename.js"></script>
<script src="js/anotherone.js"></script>   
<script>/* may have some internal code */</script>       
</body>
You can use the w3schools editor to try and test your code.
